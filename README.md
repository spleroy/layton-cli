# layton-cli

A basic Python CLI toolkit to help with CTF forensics challenges. Wraps `file`, `exiftool`, `strings`, and `binwalk`, and prints possible patterns found in the output.

## Usage

```bash
python3 layton.py <file>
```

## Requirements

- Python 3
- `file`, `exiftool`, `strings`, `binwalk` installed and on PATH
